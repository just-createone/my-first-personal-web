《成功，动机与目标》总结

1.”做到最好“不够好​

  • 将目标具体化，最好限定时间，量化每一次行动的具体任务，这样才会让自己持续的做下去。
  • 将目标困难化，目标设置可以有一点难度，让自己需要使使劲努努力才能够实现，而不是每天的重复打卡，对于成长没有一点帮助。
  • 为什么或是什么，当做一件事想要获得更多的动力，就用”为什么“的思维模式去思考这件事的意义，当想要行动起来就用”是什么“的思维模式去思考一件事情，促成自己的行动。
  • 请考虑价值和可行性，当目标为短期目标的时候要多思考可能带来的好处是什么？当目标偏向于长期的时候，要多思考具体的行动计划是什么？
  • 正面思考，但不要轻敌，大多数值得奋斗的目标都需要时间和努力，因此不要因为短暂的失败蒙蔽了自己的双眼，而是要思考这项计划的可行性，以及后面可能出现的结果。
  • 运用心里对照法制定目标，当制定目标的时候除了想到可能的成功，还要思考可能的失败，做出相应的反应措施。

2.你还能变聪明吗？​

  • 懂得什么影响你。如果想要做出好的选择，要明白什么在暗中影响着自己是很重要的。
  • 懂得如何判断自己的能力。问问自己的能力有多强？是否足以支撑自己的信念？很多时候认为不行是因为自己的能力不够强！
  • 拥抱改变的机会。虽然自己的能力有限，但是相信自己的能力可以发展出来与相信自己的能力是一成不变的有着本质的区别，前者更有机会改变，后者只能等运气。
  • 营造正确的环境。环境是改变自身的一大因素，正确的环境有利于自身的发展。
  • 利用触发点进入潜意识。想要充满动力，就必须要在环境中充满提示和触发点，让自己在追梦的路上能够保持充足的动力以及强大的执行力。

3.你注重出色表现还是成长进步？​

  • 表现还是进步？意在表现与意在进步的目标之间存在着本质的区别，当我们的注意力放在表现上的时候，就会在意外界的评价，影响自身对目标的追求。在意进步的时候就不会在乎评价，遇到负面消息或者失败的时候也只是去思考如何改进。
  • 聚焦表现带来成绩，当事情不是很复杂的时候，聚焦表现可以带来和那后的结果。但是无法处理复杂的事情。
  • 聚焦进步提升成绩。当事情无论变得多复杂，追求进步的人能把失败和困难化为力量，不断解决问题，提升自己的能力。
  • 聚焦进步，享受旅途。进步让人更加专注于过程，并在解决问题的过程中找到乐趣。
  • 聚焦进步，抵抗抑郁。个人成长的目标比自我证明的目标更能使人从容有效的面对抑郁和焦虑，失落的心情能激励他们行动、解决问题。
  • 聚焦进步，收获更多。进步注重改进，并且沉迷改进，没有时间去忧伤，最终收获的也只有成长和进步。

4.被爱与安全​

  • 进取为了获得，防御为了不失去。进取是为了收益最大化，防御是为了避免损失。
  • 乐观精神适合进取目标。自信和乐观是进取型人格的燃料，有助于进去型人格更进一步的发展。
  • 乐观精神不适合防御目标。过度的乐观可能会对目标产生影响，降低人的警惕性，忽略一些重要的点，进而造成损失。
  • 进取使人热血沸腾，防御使人如释重负
  • 进取目标偏爱风险
  • 防御目标偏爱谨慎
  • 合适的策略。在识别出自己目标的种类之后，适当的采取进取和防御的目标有利于自己的行动，正确的行动会带来正确的结果，让人愉悦。
  • 视情况而定。不要采取固定的思维来界定目标，有时候相同的目标在不同的阶段会变成不同的性质，需要我们能够识别出来，进行恰当的转化，采取合适的策略。

5.不闪光的金子​

  • 人类的三种基本需求：关联感增强人际关系；胜任力开发新技能；自主权反应出热情之所在。
  • 闪光的不一定是金子。追求表现型目标会让自己的幸福度下降，变得不那么快乐。
  • 内在动力可以燎原。自由选择的目标能够生成内在的动力，且不受外界因素的影响。
  • 自主权是动力的燃料。

6.实现目标需要思维转换​

  • 事情容易的时候选择进取目标。多聚焦于成功带来的好处。
  • 没动力时，多想想大局。
  • 事情困难时，目标要具体。多制定具体的实施步骤，量化自己的目标内容，帮助自己实现自己的目标。
  • 诱惑横行时，多用为什么思考。多想想为什么自己要做这件事情，意义在哪里，想到这些行动的理由，就不会被别的东西所干扰。
  • 如果你需要速度，选择侧重于收益的进取目标。
  • 如果你需要准确，选择避免损失的防御目标。
  • 如果你需要创造力，选择进取目标
  • 如果你希望奋斗的过程是愉悦的，选择进步目标，同时确保目标为自主选择。
  • 如果你想拥有真实、长久的幸福，选择满足关联感、胜任力、自主权这三个基本需求的目标。

7.帮助他人实现目标​

  • 使目标个人化。拥有自主选择权的目标给人一种掌控感，让人能够更好的投入到目标的执行和实现过程中去。
  • 公众承诺。当没有个人选择的余地时，可以让他人立下契约。公开承诺能够增强他们的动力，没人喜欢违背诺言。
  • 运用适当的触发因素。追求的目标大都是在潜意识中触发的，需要自己不断的努力增加触发点，帮助自己实现自己的目标。
  • 理念框定。框定出任务的性质，能自然的采用合适的目标。
  • 让目标蔓延。当追逐某个目标的时候，只要认真专心的做好一件事情，那么这件事情就会产生极强的感染力，让人们对次产生向往之情。

8.行动之前的准备——了解目标破坏者。​

  • 行动最重要。
  • 抓住当下。
  • 知道做什么
  • 拿起盾牌。抵抗诱惑，不要被一些无关紧要的东西干扰。
  • 了解现状。客观的认清自己的现状，做出符合时机的选择判断，聚焦于当下，制定出可执行的策略，实现自己的进步改变。

9.如果......就......的力量​

  • 制定计划。计划是一切变化的基石，只有具体的计划和行动准则才能够让自己的目标落地，不会变成空谈。
  • 决定具体行动。量化自己的每日任务。
  • 决定时间和地点。关联外界的一切事物，帮助自己更好的实现目标，一旦进入到某个场景，就必须要去做与目标相关的事情。
  • 整理出”如果......就......"计划。把计划的内容讲给自己听，变成一句详实落地的语句。
  • 瞄准障碍。想想达标路上可能出现的障碍，并且想好应对措施，帮助自己更好的进入到目标执行过程中去。

10.顾此失彼的自制力——“自制肌”的疲劳和保养​

  • 不进则退。自制肌就像肌肉一样，不锻炼就会萎缩。
  • 启动自制力。接受一些小的挑战，可以锻炼自己的自制力，慢慢的强化自己的自制能力。
  • 休息必不可少。不要用力过猛，不要消耗过大，要学会适当的放松自己。
  • 来点甜食。自制力与血糖有关，当自己体内的血糖升高时，自制力也会相对变得更强一点。
  • 别冒无畏的险。无论自己的自制肌有多强，都需要将其用在最重要的事情上面。别在没有用的地方过度消耗自己的自制力。

11.聪明的乐观者——让信心切合实际​

  • 适度乐观时有益的。乐观精神能带来很多好处，能增强动力。
  • 有时乐观时危险的。乐观者更有可能选择让人自我感觉良好的心态对应挫折，但是容易忽略需要改善的地方。
  • 了解乐观的种类。对于无法掌控的情况盲目乐观是不可取的，对于自主权在自己手上的事情乐观是可取的，他能帮助人们用进取的态度去做事。
  • 保持清醒。基于现实的乐观是可取的，因为对现实中的事情有了一个清晰的认识，可以较好的掌控自己的步骤和计划。
  • 如果不真实，使他变真实。如果在乐观的过程中发现自己的目标不切实际，那么就尝试着找出可能存在的障碍，分析解决这些障碍需要什么样的步骤和代价，然后做出符合实际的决策行动。
  • 把注意力从能力上转移。多聚焦于努力和行动，而不是当前的能力，当前的能力是会随着自己不断的努力改变的。
  • 翻开过去的一页。回想自己过去的成就是保持乐观的一种方式。
  • 别构想成功的画面。想到实现目标后的成功使不切实际的画面，没有什么指导意义，想象自己在实现目标过程中所付出的行动和努力，以及解决问题的方式和态度，这种形式的乐观使有利于自己达成目标的。

12.打得了持久战——懂得坚持，实时放弃。​

  • 你坚韧吗？愿意为目标长期投入的人更容易成功。
  • 坚韧起来。进步和自主是一个人变得坚韧的有效办法。
  • 怪你不努力，不怪你笨。如果相信达不成目标是因为缺乏能力，那么就会陷入自我否定，相反认为能力是可以塑造的，没有达成目标只是因为自己不够努力，这样的想法会让自己重视勤奋、计划、坚持，更容易让自己成功的概率提升。
  • 你无法拥有一切。认为自己能力不够而放弃目标是不理智的选择，但这并不意味着你要坚持所有的目标，适当的放弃一些不那么重要的，将有限的精力投入到重要的事情中去才是可取的办法。
  • 有的代价不必付出。当你能实现很想要的目标但是代价太大的时候，放弃也是完全没有问题的选择。
  • 旧的不去，新的不来。放弃有时候也很难，选一个新的符合实际的代价不那么高的目标代替旧的目标才是明智的选择。

13.给予适当的反馈——批评和表扬的策略​

  • 说实话。要客观的评价自己和他人，这样会有一个更加清晰的认识，从而做出合理的决策。
  • 正面实用的建议。发现问题及时提出来并且要及时解决。
  • 真诚的赞美。将表扬贴合实际，结合具体事件和成绩做出自己的表扬。
  • 表扬请对事不对人。
  • 避免比较。当把注意力放在比较上时，就不可避免的会变成表现型人格，对于自己的成长没有好处，实在要比，也要和过去的自己相比。
  • 得到表扬不是目的。别让表扬与奖励削弱了自主权。
  • 务实。将所有的注意力放在贴合实际上面去，这样所有的判断和行动就会落地，而不是空想，更不容易引发焦虑消耗自己的精神力量。
